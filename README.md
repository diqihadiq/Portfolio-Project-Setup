## What This Project Is

This repository documents original research into how practitioners are using AI to produce, optimize, and scale SEO content in 2025-2026. The goal is to synthesize insights from 10 leading experts into a practical playbook - not just a summary of what people say, but a working framework based on what they actually do.

The research covers:

- How AI is changing content production workflows
- What still requires human judgment vs. what can be automated
- How topical authority and semantic SEO intersect with AI tools
- What E-E-A-T means for AI-generated content
- Practical tools and pipelines practitioners are using right now
