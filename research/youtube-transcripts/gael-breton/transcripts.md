[On MCP servers — the big opportunity:]

Essentially an agent is going to do low-level SEO tasks for you if it has access to an Ahrefs MCP server. The amount of workflows you can build is unlimited. And the ability for small players to scale up operations, deliver more value without more human workflows, is going to be huge.

## Key Themes from These Transcripts

- Google Workspace (Gemini built into Gmail, Sheets, Docs, Meet) is more practical than ChatGPT because it eliminates context-switching and is already connected to your data
- Gemini Notetaker in Google Meet is a free alternative to tools like Otter.AI — and the real value is using that transcript context in other Google apps
- AI in Google Sheets can analyze data in natural language, visualize it, write formulas, and mass-categorize data using the AI() function
- Creating documents (proposals, emails) from call transcripts with @-referencing in Google Docs is one of the most practical daily workflows
- Building a personalized "email replier" Gem trained on your own writing style is a high-leverage one-time setup
- Notebook LM is underused — works as a personal research engine over YouTube playlists and sales call transcripts, not just for learning
- For newsletter creation: write rough notes + system prompt with real examples → AI drafts → human review and feedback loop → final edit. ~2 hours per week
- The original idea still has to come from you — AI handles formatting, structure, and polish. Going too far with AI kills authenticity and social reach
- Gemini Deep Research + audio overview = custom podcasts on niche topics you care about, consumable during gym sessions
- MCP servers are the next frontier: give AI agents access to tools like Ahrefs to run recurring SEO tasks autonomously and make low-level decisions
- SEO is not dead — LLMs are grounded in search (Bing/Google), so creating content, brand links, and thought leadership still determines who gets cited
- Make.com (and N8N) connect AI steps into full automations — but real-world quirks (video uploads, API limits) still require human touchpoints
